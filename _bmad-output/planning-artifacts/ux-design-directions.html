Comparison — All Directions

Side-by-side evaluation of all four design directions against Credai's key design goals.

Criteria A — Structured Authority B — Clean Minimal C — Bold Dashboard D — Staff Review
Institutional trust signal ●●● Green header, official feel ●●○ Clean but subtle ●●○ Modern, less official ●●● Admin authority
Form density (efficient/moderate) ●●● 3-col grid, card sections ●●○ Slightly airier ●●● Dense, sidebar saves space N/A (review surface)
Step visibility ●●○ Horizontal rail (can overflow) ●●● Dot progress + % bar ●●● Full sidebar list — best N/A
Document upload UX ●●● Chip with View/Remove inline ●●● Drop zone + uploaded chip ●●● Upload row list N/A
Staff auditability ●○○ Not applicable ●○○ Not applicable ●○○ Not applicable ●●● Checklist + inline docs
Familiarity to Indian users ●●● Closest to bank/GSTN portals ●●○ Modern but less familiar ●●○ Enterprise SaaS feel ●●● Admin tool feel
